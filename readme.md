# Template
This is readme.md template

## Prerequisites

1. template
2. [template](https://example.com/)
3. template

## Image from source

1. Clone this repo

    ```shell
    git clone https://github.com/TechDepSut/readme_template.git
    ```

2. Build image 

    ```shell
    docker build -t template .
    ```

3. Run container based on your image

    ```shell
    docker run -d template
    ```
